# On The Go

A modern Flutter e-commerce application designed for seamless product discovery and order management.

## Features

- **Store & Browsing**: Explore a wide range of products with categories and best-selling brands.
- **Order Tracking**: Real-time updates on your ongoing orders.
- **Quotes**: Request and manage bulk quotes easily.
- **Cart Management**: Simple and intuitive shopping cart experience.
- **User Profile**: Manage account settings and preferences.

## Tech Stack

- **Framework**: Flutter & Dart
- **Navigation**: GoRouter & Curved Navigation Bar
- **UI/UX**: Google Fonts, Carousel Slider, Flutter SVG

## Getting Started

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/yourusername/on_the_go.git
    cd on_the_go
    ```

2.  **Install dependencies**:
    ```bash
    flutter pub get
    ```

3.  **Run the app**:
    ```bash
    flutter run
    ```

## Requirements

- Flutter SDK: `^3.10.1`
- Dart SDK: Compatible with Flutter version
